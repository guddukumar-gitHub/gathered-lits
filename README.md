# GatheredLits - Book Store Website

A beautifully designed book store website featuring a curated collection of literary works.

## 🌐 Live Deployment

**View the live website here:** [https://gatheredlits.netlify.app/](https://gatheredlits.netlify.app/)

## 📋 Project Overview

This is a simple yet elegant book store website built with HTML5, CSS3, and Bootstrap 5. The website showcases a diverse collection of acclaimed books across multiple genres.

## 📄 Pages

1. **Home** - Landing page with featured books and highlights
2. **Books** - Complete collection of books with filtering options
3. **About** - Information about the website, mission, vision, and team
4. **Contact** - Contact form, FAQ, and location information
5. **Login** - User authentication

## 🎨 Features

- Responsive design that works on all devices
- Bootstrap 5 framework for modern UI
- Genre filtering capabilities
- Professional navigation and footer
- Clean and user-friendly interface
